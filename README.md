# Phishing Link Detector

## Overview
This project is a Phishing Link Detector that identifies potential phishing links and helps users avoid malicious websites.

## Features
- Detects phishing URLs
- Provides a confidence score for each detected link
- User-friendly interface

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AdamHarish25/Phising-link-detector.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Phising-link-detector
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application and input the URL you want to check. The system will analyze the link and provide feedback on the likelihood of it being a phishing attempt.
